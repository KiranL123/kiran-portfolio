## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project directory
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

The application will open at `http://localhost:5173`

### Build

Create a production build:
```bash
npm run build
```

### Preview Production Build

Preview the production build locally:
```bash
npm run preview
```

## Customization

### Update Your Information

1. **Personal Details**: Edit the text in `src/components/Hero.jsx` and `src/components/About.jsx`
2. **Projects**: Modify the projects array in `src/components/Projects.jsx`
3. **Skills**: Update the skills categories and levels in `src/components/Skills.jsx`
4. **Contact Info**: Change contact details in `src/components/Contact.jsx`
5. **Colors**: Adjust the gradient colors in the CSS files to match your preference

### Color Scheme

The default color scheme uses a purple gradient (`#667eea` to `#764ba2`). To change this:
- Search for these hex values in the CSS files
- Replace with your preferred colors

### Adding Images

Replace the placeholder elements with your actual images:
- Profile picture in `About.jsx`
- Project screenshots in `Projects.jsx`
- Add images to the `src/assets` folder and import them in the components
